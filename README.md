# 多益訓練場

單字卡 SRS、Part 5/6/7 練習、錯題本、英美發音的多益備考 App（單一 HTML 檔，離線可用）。

## 學習紀錄與同步

- 紀錄存在**每台裝置的瀏覽器本機**（localStorage），不會自動跨裝置同步。
- 要在手機與電腦之間對齊：在總覽 →「資料」按「下載備份檔」，到另一台用「合併匯入」即可（合併是冪等的，來回同步不會亂）。

## 佈署

本頁由 GitHub Pages 託管（Settings → Pages → Deploy from a branch → main / root）。
更新內容時重新上傳 `index.html` 覆蓋即可，網址不變、各裝置紀錄不受影響。
